# @i18n-micro/utils

Framework-agnostic helpers shared across **nuxt-i18n-micro** and `@i18n-micro/*` packages: URL/path parsing, cookie helpers, `defineI18nRoute` extraction, locale path resolution, merge utilities, and related pure functions.

The package avoids Nuxt/Vite/Nitro-specific imports so it can be bundled in any environment.

## Install

```bash
pnpm add @i18n-micro/utils
```
