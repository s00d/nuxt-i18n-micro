# Composables

## Available Composables

- [🛠️ `useI18n` Composable](./useI18n.md) - Main composable for translations
- [🌍 `useLocaleHead` Composable](./useLocaleHead.md) - Composable for SEO meta tags
- [📄 `useI18nHead` Composable](./useI18nHead.md) - Page-level SEO overrides (articles, CMS, custom hreflang)
- [📍 `useI18nLocale` Composable](./useI18nLocale.md) - Centralized locale state, cookies, and sync utilities
