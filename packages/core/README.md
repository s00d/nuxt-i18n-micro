# @i18n-micro/core

Translation, formatting, and routing helpers shared by **nuxt-i18n-micro** and the `@i18n-micro/*` integration packages. Most apps depend on this transitively; install it directly only if you build custom tooling on the same primitives.

## Install

```bash
pnpm add @i18n-micro/core
```
